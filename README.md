# AI Resume Analyzer

A modern AI-powered Resume Analyzer built using Streamlit and Python.

## Features

- Resume Parsing
- Skill Extraction
- Resume Score Analysis
- ATS-style Recommendations
- Course Recommendations
- PDF Resume Upload
- User Dashboard

## Tech Stack

- Python
- Streamlit
- NLTK
- PyResParser
- MySQL
- Pandas

## Installation

```bash
git clone https://github.com/sindhu737/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer

python -m venv venvapp
venvapp\Scripts\activate

pip install -r App/requirements.txt

streamlit run App/App.py
```
